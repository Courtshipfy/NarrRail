# NarrRail AVG 插件任务计划书（WBS 细分版，可勾选）

> 版本：v0.2（2026-03-16）
> 适用仓库：`I:\NarrRail`
> 目标：交付可在 UE5.7 中生产使用的完整 AVG 插件（对话流程、故事线驱动、剧本配置、导入导出、编辑器内编辑）

## 1. 使用规则（监督与打勾）

- 勾选规则：仅当“DoD 达成 + 测试通过 + 证据提交”才可从 `[ ]` 改为 `[x]`。
- 父任务勾选规则：父任务仅在其所有子任务都已 `[x]` 后才可勾选。
- 最小监督证据：提交记录（commit/PR）、测试记录、演示证据（截图/录屏）。
- ID 规则：所有任务使用唯一 ID，格式为 `NR-模块-主项-子项`，例如 `NR-RUN-002-04`。

## 2. 里程碑总览

- [ ] `M1` 运行时核心可用（完整剧情可跑）
- [ ] `M2` 剧本配置与导入导出可用（可往返）
- [ ] `M3` 编辑器内可视化编辑可用（可用于日常制作）
- [ ] `M4` 调试、自动化测试与质量门禁可用
- [ ] `M5` 文档、示例、打包与发布流程可用
